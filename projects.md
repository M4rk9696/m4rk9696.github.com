## Tiddier

A small extension I use to make webpages readable. Injects this into `body`
```
body {
  width: 80vw;
  margin: auto;
  line-height: 1.7;
  font-size: 1.19em;
  font-family: "Open Sans", sans-serif;
}
```

[Source](https://github.com/Mark1626/Tiddier)
